📘 第一份作業：地球物理概論 Playground

主要內容

  • 地球分層 — 地殼、地函、地核；解釋地函對流、地核生地磁場的重要性。
  • 重力與大地測量 — 地球不是完美球體、利用重力異常找地下構造。
  • 地震學與地球內部 — P 波、S 波差異，P‑S 時差用於估算震源距離並推演地核狀態。
  • 熱流、海底年齡與電性 — 新生海洋地殼熱、遠離洋脊越老越冷，支持板塊擴張理論。
  • 地磁與古地磁 — 岩石冷卻時記錄當時地磁，海底磁條帶提供板塊漂移證據。

特色

  • 將抽象理論轉化為易於理解的「生活例子 + 記憶口訣 + 小互動 Quiz／計算器」
  • 每一章節都有核心概念、為什麼重要、生活連結與記憶口訣 — 適合初學者快速入門

🌐 第二份作業：Free‑Air Gravity 異常分析

理論背景與方法

  • 進行自由空氣校正 (free‑air correction)，把測站重力還原到海平面基準面。
  • 計算 free‑air 異常 (free‑air anomaly)，反映地下密度異常 (暗示可能的低密度區如鬆散層、沉積盆地、空洞…)。
  • 若進一步扣除地表質量／地形影響，可做布格校正 (Bouguer correction)，得到更貼近構造密度異常的結果。

本次結果 (範例)

項目數值 / 說明
觀測重力 g_obs範例值 (依作業)
理論重力 g_t依地理位置與緯度計算
自由空氣校正 (FAC)依海拔高度計算補正值
free‑air 異常 Δg_FA異常值 (正或負) → 判斷地下密度異常
布格校正 (若有)扣除地表質量後異常值，較能反映地下構造

應用與限制

  • 適合作為初步判讀地下構造 (盆地、低密度沉積層、空洞…) 的工具。
  • 若地形起伏大或地形質量差異大，需做進一步布格 + 地形修正 (terrain correction) 才能提高準確性。
  • 僅靠重力異常無法唯一判斷構造，建議搭配地震、地質、鑽探或其他資料做綜合分析。

🗺️ 第三份作業：板塊邊界地圖 (PyGMT)

目的與方法

  • 使用 PyGMT（基於 GMT 的 Python 套件）繪製全球或區域板塊邊界地圖。
  • 設定 region／projection／底圖 (coastline, land/sea, water, bathymetry...)，並把板塊邊界資料繪製為線。
  • 產出高解析度地圖 (PNG 或 SVG)，方便用於報告、簡報、視覺化分析。

可呈現成果

  • 全球板塊邊界分布圖 — 便於辨識主要板塊與構造帶 (如中洋脊、俯衝帶、斷層帶…)。
  • 若針對特定區域，亦可製作區域／局部放大圖，更清楚對應地質背景、地震或火山分布。

潛在用途

  • 做為後續地震、板塊運動、構造演化研究的背景地圖。
  • 視覺化成果方便整合到報告／簡報／論文中。

🔨 第四份作業：校園震測實驗報告

實驗內容

  • 在校園 (黑森林 → 圖書館後方) 架設 24 支地震儀 (geophone)，進行淺層地震勘測 (敲擊鐵板產生震源)。
  • 收集直達波 (直接從地表傳遞) 與折射波 (經地下介面折射再傳回地表) 的走時資料 (travel‑time)。
  • 利用臨界折射 (critical refraction) 理論 + 走時圖分析，推估地下分層與地層厚度 (淺層結構)。

資料處理與結果

  • 從走時圖中判斷直達波與折射波的斜率 → 得到各層波速。
  • 利用速度與到時差估算淺層厚度 (平均值)，作為該校園地區近地表結構的初步推估。
  • 結果顯示地表上層為鬆散層 (速度低)、下層為較緻密地層 (速度高)、厚度為數公尺等級 (取決於你實際量測與計算)。

學習與反思

  • 從教室理論走到校園實作：親手架儀器、量測、分析、報告 — 對地球物理有更實感的理解。
  • 資料處理中發現：震測不只是公式、也涉及到現場操作品質 (震源、訊號品質、雜訊…)。真正的地球物理研究，不只是推理，更是工程與判讀。
  • 這次經驗對後續更深的地震、構造或地質研究很有幫助，也為期末報告建立實作基礎。
